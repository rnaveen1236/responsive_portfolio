# 🌐 Responsive 3D Portfolio Website

A modern **3D animated responsive portfolio website** built using **HTML, CSS, and JavaScript**.

This portfolio showcases personal details, projects, skills, education, and contact information with smooth 3D cube transitions and interactive UI effects.

---

## 🚀 Project Overview

This portfolio website includes:

* 🏠 Home Section
* 👤 About Section
* 📄 Resume Section
* 💼 Portfolio Section
* 📬 Contact Section
* 🔄 3D Cube Page Transition Animation
* 📱 Fully Responsive Design

The website uses a rotating cube layout for section navigation, creating a unique interactive experience.

## ✨ Features

### 🔹 3D Cube Navigation

* Each section is placed on a 3D cube face
* Clicking navbar icons rotates the cube
* Smooth animated transitions

### 🔹 Resume Section Tabs

* Projects
* Skills
* Education
* Dynamic tab switching using JavaScript

### 🔹 Portfolio Section

* Project previews
* GitHub repository links
* Hover effects with overlay

### 🔹 Contact Form

* Integrated with Web3Forms API
* Allows users to send messages directly

### 🔹 Responsive Design

* Works on:

  * Desktop
  * Tablet
  * Mobile devices
* Media queries included
